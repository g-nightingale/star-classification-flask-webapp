About

I built this webapp as a means of learning how to use Flask & Python, with integrated ML elements and Plotly visualisation.

The main functionality of this webapp is to predict a star type given input data including temperature, luminosity and radius. The webapp also has functionality to inspect the underlying data via interactive Plotly visualisations.

Hertzsprung-Russell diagram

The Hertzsprung-Russell diagram is a visual representation between the luminosities and effective temperatures of stars and can be used as a guide for classification. This method of visualising star classes was developed in 1910 by Ejnar Hertzsprung and Henry Norris Russell.
